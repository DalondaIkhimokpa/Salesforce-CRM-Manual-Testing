CRM-001: Certification Reminder Logic

Priority: High | Type: Functional | Status: Designed

Objective: Verify system sends reminders 14 days before certification expiry

Steps:

  1. Set test certification date to today + 13 days
  2. Trigger daily batch process
  3. Verify reminder email is generated

CRM-002: Picklist Dependency Validation

Priority: Medium | Type: Validation | Status: Designed

Objective: Confirm dependent picklists show correct values based on controlling field

Steps:

  1. Select "Technical" in Department field
  2. Verify only technical roles appear in Job Role picklist
  3. Repeat for other department values
